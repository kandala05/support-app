## Why Support App?

Traditional issue tracking captures problems but leaves resolution to human knowledge. Support App actively helps users solve issues through intelligent documentation surfacing and AI assistance.

<CardGrid stagger>
  <Card title="Smart Documentation" icon="document">
    As users describe issues, relevant documentation surfaces automatically. No more searching - the right docs find you.
  </Card>
  <Card title="AI-Powered Assistance" icon="rocket">
    Claude-powered chat interface understands context, suggests categories, and guides users to solutions.
  </Card>
  <Card title="Closed-Loop Learning" icon="approve-check">
    Track which docs help resolve issues. AI identifies patterns and suggests documentation improvements.
  </Card>
  <Card title="Diataxis-Organized" icon="open-book">

## Product Phases

| Phase | Focus | Description |
|-------|-------|-------------|
| **V0** | Documentation | This site - comprehensive docs with interactive prototypes |
| **Phase 1** | Basic Tracking | Multi-step issue submission, plane categorization |
| **Phase 2** | Smart Sidebar | Real-time doc suggestions as users type |
| **Phase 3** | Chat Interface | Conversational AI for issue creation |
| **Phase 4** | Analytics | Insights dashboard, pattern detection |

## Tech Stack

- **Frontend**: Next.js 15, React, TypeScript, Tailwind CSS
- **Database**: PostgreSQL with Drizzle ORM
- **AI**: Anthropic Claude via Vercel AI SDK
- **Infrastructure**: AWS via SST (RDS, Cognito, S3, CloudFront)
- **Documentation**: Astro + Starlight
